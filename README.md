# Picsite
この開発に参加する方は、以下の用に名前をサインインしてください。
このレポジトリをローカルにクローンして、READMEを編集。
その後PRを送ってください。

```shell
$ mkdir picsite
$ cd picsite
$ mkvirtualenv picsite -p python3.7
#git clone this repo
$ cd picsite
$ pip install -r requirements.txt
$ ./manage.py migrate
$ ./manage.py runserver
```
